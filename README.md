# Banana Island Emergency Phone Locations

This project visualizes proposed locations for blue light emergency phones within Banana Island, Lagos, Nigeria. Utilizing Python libraries such as OSMnx, GeoPandas, Contextily, and Matplotlib, the script generates a map highlighting potential installation sites to enhance public safety infrastructure.

## Features

- **Boundary Visualization:** Displays the geographical boundary of Banana Island.
- **Road Network Mapping:** Illustrates the road network within the neighborhood.
- **Building Footprints:** Shows the locations and footprints of buildings.
- **Proposed Emergency Phone Sites:** Highlights suggested locations for blue light emergency phones.
- **Basemap Integration:** Provides contextual background using a subtle basemap for clarity.
- **Custom Annotations:** Includes descriptive annotations detailing the safety initiative.
